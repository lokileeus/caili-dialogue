# 真实备婚聊天记录：彩礼拉锯全过程
（时间：2025年2月，微信对话，无删减）


##故事背景
女方名字叫牛田，男方名字叫黎蔡君。

25年春节，黎蔡君和牛田通过相亲认识，他们2人是同村的，相亲的方式是传统的通过媒人介绍。双方家长和双方当事人一起见面，黎蔡君今年33岁，而牛田今年28岁。双方见面见面后，牛田的母亲感觉是特别不满意，朋友黎蔡君说：“当时气氛很尴尬的，主要是他不想去相亲，但是她妈妈又答应媒人了。”

回到家后，女方通过了朋友的微信好友。朋友想着还是主动聊一下吧，既然她都同意加好友了，媒人说的似乎也有点道理然后就开始聊了起来。

第二天朋友主动给女方发信息，女方回复得很慢。朋友以为女方和之前得相亲女一样，也是被逼相亲的，朋友也如释重负，也就故意表现得很冷淡。女方得消息故意过了2-3小时回复但这个时候女方仿佛又回复得很快。朋友妈妈问聊的怎么样，年初一初二要不要去她们家拜年。所以男方在微信里问女方去她们家拜年，这个时候女方却极力推辞。男方只好作罢。这个时候男方不知道女方到底是什么意思。

过了正月初五男方准备回深圳工作了，女方还要在家待到正月初十，十一，十二的样子。

女方和她母亲坐了一天一夜的长途大巴，第二天上午9点到深圳，那天男方在上班。男方妈妈问男方怎么不去接她们。男方妈妈估计想的是，想讨好女方，让女方家留下好感。但男方觉得这是无理的讨好。

因为她们乘坐了一天一夜的车，在卧铺车上，吃也没吃好，睡也没睡好。所以白天她们在睡觉。下午女方表示明天自己要上班了，而且这个周末没有休息要连上9天，所以趁今天有空可以出来见面。男方很开心，毕竟相亲了那么多女孩这是第一次女方主动邀约，而且还是女方在乘坐一天一夜刚到深圳后就发起邀约。就算是男孩子追女孩子也没有这么主动的吧。所以男方提前购买了电影票，并叫外卖送了一束鲜花到男方公司，约定下班后在地铁口回合。

下班后男方捧着鲜花，到了地铁口。由于离第一次见面已经过去十多天了。所以男方对女方的外貌有点模糊不清了，但还是在地铁口认出了女方。看完电影后已经11点左右了，地铁也停运了。男方提议让女方把地址给他，他给女方叫车送她回去。“

牛田说：“不用了，我自己叫车吧”

男方瞬间脸上挂满了失望。等女方的车来了后目送女方上车离开，男方也自己走回家了。男方心里想着，可能是女方没看上他吧。决定不再主动聊天了。第二天女方又主动聊天，男方故意不回复。或是延迟回复。

黎蔡君说：“昨天晚上我生病了，发烧所以现在才醒来”

牛田又说：“是她的不好，黎蔡君生病了，她也没看出来。”

黎蔡君仿佛又燃起希望。又主动聊天，但是对于女方没有给他地址还是耿耿于怀。

再过几天快到情人节了，黎蔡君想着正是表现得机会。于是接着这个机会再次向牛田要地址。

黎蔡君：“把你的地址发我一下”

牛田：“嗯？怎么啦？”

黎蔡君：“后天不是2.14给你买点花束，小蛋糕什么的”

牛田：“花到人不到？送花是想表达什么呢？”

黎蔡君：“就是表达交往的意思啊”

女方明知故问，想让男方表态。这是缺乏安全感，想得到男方心理确认。通过反问，让自己站在道德高地，让男方既送花又要人到现场。

黎蔡君：那你要我过去，你要先把地址告诉我啊，你不告诉我，我怎么过去？晚上下班我搭地铁，如果带花的话地铁很拥挤，会把花挤烂的”

牛田：“ 等一下我在外面，刚吃完饭，你吃饭了吗？”

黎蔡君：“我吃完了”

牛田：“你又不来只送了礼物，我会想着你是不是只是礼貌性的问侯一下，后来你解释我就懂了。那天我也不知道什么时候下班，你的心意我收到了，下周末再约吧。”

黎蔡君：“好的。”

对话解读

当男方表示自己会过去，但是想把花快递过去后，女方使用了缓兵之计。用我在外面搪塞。过了一小时后女方又主动找男方。

女方觉得不给地址好像自己说不过去，然后找了一个理由。心意收到了，我也不知道什么时候下班，下周再约。这在男方看来就是拒绝追求的意思。

过了10分钟后，牛田主动联系黎蔡君

牛田：“要不要去爬山？”

黎蔡君没有回复牛田，牛田继续发信息

牛田：“失联啦？”

黎蔡君：“抱歉我们不合适。”

牛田：“怎么啦？”

黎蔡君：“感觉你又点装。”

对话解读

男方态度冷淡，女方又开始撩拨，问男方要不要去爬山。男方直接回复我们不合适。并直言你有点装。

牛田：“啊？因为你送花我没有收？”

黎蔡君：“就这样吧说多了也没什么意思了。”

牛田：“不明白，我希望我们能好好沟通一下啊，我不想错过，如果是因为昨天的事我有不对的地方的话我道歉，但是我确实是没有理解你的用意，哪有送花人不到的哇？你后来不是解释了吗。我就想通了啊，然后如果说见面的话来回就两个小时了时间去了一大半，你能送我礼物我当然很开心啊，可是比起礼物我更希望是见面啊。所以我想着周末见面啊。”

黎蔡君：“毕竟我们是相互了解的过程，相亲就是这样的。还没开始，所以没有什么错过不错过的。”

牛田：“那我们重新开始嘛，我明天下班去找你。”

黎蔡君：“其实 相处10多天，也应该没有很深的感情。你可能只是不甘心我说不台适。其实及时止损比较好。越到后面越发现不合适沉默成本越大，越难决定。”

牛田：“感情都是慢慢培养的啊，你要说不甘心我也不知道如何回答，你是我好不容易遇到的一个心动的，我不想放弃。我不知道你说的不合适指的是什么？如果你要说工作能力我确实不如你，你别气了嘛和好呗。我以为是我们之间出现了不必要的误会影响了我们，我也表达了我的想法遭从你内心的决定，我等你的答复。”

过了很久黎蔡君没有回复牛田

牛田：“温馨提示：你已经超过半天没理我啦！再不理我就变成，狗不理了。理一下嘛 我是真的真的诚心诚意的道歉，不该辜负你的好意。”

黎蔡君：“那你把地址给我。”

对话解读

经过女方得不断道歉男方还是接受了女方得道歉。并第一时间要了女方的地址。这也是男方对女方的服从性测试。男方觉得女方不给他地址是还不信任他，聊了10多天见了2次面还不信任他，那么没必要再聊了。

牛田很爽快的把地址给了黎蔡君，并约好周日去爬山，并且确定了恋爱关系。
