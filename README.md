# 彩礼对话实录

这是一个 2025 年真实微信聊天记录，记录了一对普通备婚情侣从甜蜜到冲突再到分手的完整过程。

## 内容概述
- **时间**：2025年2月左右
-- **主题**：彩礼、三金、房子、父母参与、提亲等备婚拉锯
- **核心冲突**：女方高情绪敏感 + 安全感需求 vs 男方现实压力 
- **结局**：删微信、分手
